Simple Calculator

Calculadora simples, grátis e com interface limpa.

Para realização de operações simples de matemática. Este projeto é para estudo e prática de Python e a biblioteca PySide6.

Ambiente de Desenvolvimento

Anteriormente o código estava sendo implementado com PyQT6, porém preferi prosseguir com PySide6. Essa escolha se deve à facilidade de utilizar a ferramenta Designer que já vem junto com a biblioteca PySide6 e também pela sua licença de uso.

Esta aplicação está sendo desenvolvida com Python 3.10.7 e PySide6 6.3.2 e são, até o presente momento, as únicas dependências que implicam no real funcionamento da calculadora. Todas as especificações do projeto se encontram no arquivo pyproject.toml, este projeto está sendo gerenciado com Poetry e para documentação estou utilizando mkdocs.

Os ícones utilizados: "Fugue Icons 3.5.6" por Yusuke Kamiyamane pela licença CC BY 3.0

Tecnologias

  1. Python 3.10.7
  2. PySide 6.3.2

Features

  • Funções aritméticas básicas.
  • Dois modos de funcionamento: Padrão e (futuramente) Científico.
  • Funções matemáticas mais avançadas: log, ln, abs, etc.

Métodos de Utilização

No momento em que se encontra essa aplicação, só temos a parte gráfica.

De toda forma, para testar a implementação é necessário estar no diretório do projeto e executar o seguite comando no cmd/shell:

python main.py
